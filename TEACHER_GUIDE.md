## Assessment Opportunities

### Formative Assessment
Monitor student progress by:
- Observing which difficulty levels students choose
- Checking hint usage (high usage = needs support)
- Reviewing accuracy percentages
- Noting common mistakes in specific problem types

### Data to Collect
The game tracks:
- Score (indicates overall performance)
- Accuracy rate (number correct / total)
- Streak count (consecutive correct answers)
- Hints used (indicates independence level)
- Badges earned (engagement metric)

### Discussion Questions After Gameplay

**Game addresses**: Step-by-step solutions show inequality-specific steps

### Misconception 2: "The direction of inequality doesn't matter"
**Game addresses**: Number line visualization shows why direction matters

### Misconception 3: "Open and closed circles are the same"
**Game addresses**: Visual number line clearly differentiates

### Misconception 4: "Math isn't relevant to real life"
**Game addresses**: 16+ real-world scenarios show practical applications

## Troubleshooting

### Students finish too quickly
- Have them try harder difficulty levels
- Challenge them to get 100% accuracy
- Ask them to explain solutions to peers
- Create their own problems

### Students are struggling
- Encourage use of hint system
- Start with easier difficulty
- Pair with stronger students
- Review step-by-step solutions together

### Technical issues
- Refresh the browser
- Try a different browser
- Clear browser cache
- Ensure JavaScript is enabled
